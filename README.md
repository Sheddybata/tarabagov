# Taraba State Unified Citizen Portal

A modern, secure e-governance platform for Taraba State Government, enabling citizens to access government services online.

## About

The Taraba State Unified Citizen Portal provides a centralized platform for citizens to:
- Report infrastructure issues and service problems
- Register births and obtain certificates
- Access land services and document verification
- Enroll in social welfare programs
- View directories of schools, hospitals, and government agencies

## Services Available

### Citizen Services
- **Report an Issue** - Report infrastructure problems with geolocation
- **Birth Registration** - Register births and obtain certificates
- **Land Services** - Access land administration services
- **Social Services** - Enroll in social welfare programs
- **Document Verification** - Verify official documents and certificates

### Directories
- **Schools Directory** - Searchable database of educational institutions
- **Hospitals Directory** - Healthcare facilities and services
- **MDA Directory** - Ministries, Departments, and Agencies

## Technology

Built with modern web technologies for performance, security, and user experience:
- Next.js 14 with TypeScript
- Responsive design with Tailwind CSS
- Secure authentication and data management
- Optimized for fast loading and mobile devices
