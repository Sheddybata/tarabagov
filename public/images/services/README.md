# Service Card Images

Save your service card images here with the following naming convention:

1. `report-issue.jpg` - For "Report an Issue" card
2. `birth-registration.jpg` - For "Birth Registration" card
3. `tagis-land.jpg` - For "TAGIS - Land Administration" card
4. `tsirs-tax.jpg` - For "TSIRS - Tax Services" card
5. `school-records.jpg` - For "School Records" card
6. `hospital-records.jpg` - For "Hospital Records" card
7. `mda-directory.jpg` - For "MDA Directory" card
8. `document-verification.jpg` - For "Document Verification" card
9. `social-services.jpg` - For "Social Services" card

## Image Guidelines

- **Recommended Size**: 800x600px or 1200x900px
- **Format**: JPG or PNG
- **File Size**: Keep under 200KB per image for fast loading
- **Aspect Ratio**: 4:3 or 16:9 works best

## Tips

- Use high-quality, relevant images
- Ensure images are well-lit and clear
- Consider the text overlay when choosing images (avoid very busy backgrounds)
- Optimize images before uploading (use tools like TinyPNG)
